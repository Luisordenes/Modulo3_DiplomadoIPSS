## Como iniciar proyecto

```
1. Clonar el repositorio:
   git clone

2. Navegar al directorio del proyecto:
    cd Api_consolas

3. Instalar dependencias:
    npm install

4. Crear un archivo .env en la raíz del proyecto con el siguiente contenido:
    PORT=3000
    MONGODB_URI=mongodb+srv://luisordenesavila_db_user:IodIYO4iEcBhKHji@clusterapiconsolas.ajebcov.mongodb.net/?appName=ClusterApiConsolas
    JWT_SECRET=ipss

5. Iniciar el servidor:
    npm run start
```
